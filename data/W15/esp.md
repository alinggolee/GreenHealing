# Week 15 ｜PBL Phase I: Integrating Design Thinking into Green Healing Project Design


## ESP Learning Focus(語言任務)

> Students will learn to:

define a clear problem(問題定義) using structured language
- **describe user needs**(使用者需求) with evidence(證據)
propose a project concept(方案概念) logically
- **connect empathy**(同理)、development(發展)、and intervention(介入)
- **justify design choices**(設計理由)

## ESP Vocabulary & Professional Usage(專業用語升級)

- **empathy**(同理)
  → “We understand users through empathy.”
- **persona**(人物誌)
  → “Our persona represents…”
- **problem statement**(問題陳述核心)
  → “This problem focuses on…”
- **user needs**(使用者需求)
  → “The user needs…”
- **intervention**(介入)
  → “This is an intervention to support…”
- **feasibility**(可行性補強)
  → “This idea is feasible because…”
- **justification**(理由說明補強)
  → “We justify this design by…”

- **ESP Functional Language**(這週最關鍵)
1️⃣ Defining the Problem(問題定義)
“Our target group experiences…”
“This is a problem because…”
“This issue is important because…”

2️⃣ Describing Needs(需求說明)
“She needs support in…”
“This need is related to…”

3️⃣ Proposing the Project(方案提出)
“Our project is designed to…”
“This project aims to…”

4️⃣ Justifying Design(理由說明最重要)
“We chose this approach because…”
“This is effective because…”
“This connects to empathy because…”

5️⃣ Evaluating Feasibility(可行性進階)
“This is feasible because…”
“We can implement this by…”

## ESP Output Frames(整合輸出)

“Our target group is…”
“She struggles with…”
“The key problem is…”
“Our project is designed to…”
“We chose this approach because…”

## TEEMI Speaking Practice(升級版)

Task Flow
- **Present**(呈現)
“Describe your persona.”
- **Define**(定義)
“State your problem clearly.”
- **Propose**(提出)
“Explain your project idea.”
- **Justify**(說明核心)
“Why is this a good solution?”

- **Problem Statement Upgrade**(幫你升級這句)
你原本:
“Adolescent girls need ___ because ___.”
建議升級👇
“[Target group] experiences [problem],
which leads to [impact],
so they need [support].”
(這會讓學生直接進入「專業等級」)

- **Design Thinking Bridge**(強化)
Students now integrate:
- **empathy map**(Week 8)
- **persona**(Week 9)
- **narrative**(Week 10–11)
- **intervention**(Week 13–14)
  → project design(Week 15)

- **Critical Thinking Focus**(這週真正核心)
Students learn:
idea ≠ solution
problem clarity = design quality
empathy must lead to action

- **Teacher Language Upgrade**(你原本很好,我幫你升級)
你原句:
“Your design should come from real understanding…”
升級👇
“Your design must come from real understanding,
not assumption.”
