# Week 14 ｜Green Healing Workshop II: Reflection and Program Design

## ESP Learning Focus(語言任務)

> Students will learn to:

- **reflect on experience**(經驗反思) and extract key insights(關鍵洞見)
identify user needs(使用者需求) clearly
- **design a simple intervention program**(介入方案)
explain program logic(設計邏輯) step by step
propose a structured plan(結構化方案) in English

## ESP Vocabulary & Professional Usage(專業用語升級)

- **target group**(目標族群)
  → “Our target group is…”
- **needs assessment**(需求分析)
  → “We identified a need for…”
- **intervention**(介入核心)
  → “This is an intervention for stress relief.”
- **activity design**(活動設計)
  → “The activity includes…”
- **outcome**(成果)
  → “The expected outcome is…”
well-being(身心健康補強)
  → “This improves well-being.”
- **effectiveness**(有效性補強)
  → “We evaluate effectiveness by…”

- **ESP Functional Language**(這週最關鍵)
1️⃣ Identifying Needs(需求分析)
“Our target group experiences…”
“They need support in…”
“This problem is important because…”

2️⃣ Proposing Intervention(提出方案核心)
“We designed this program to…”
“This activity aims to…”

3️⃣ Explaining Structure(結構說明)
“The program includes…”
“It consists of…”

4️⃣ Describing Outcomes(成果說明)
“The expected outcome is…”
“This will help participants to…”

5️⃣ Evaluating Effectiveness(評估進階)
“We can evaluate this by…”
“This is effective because…”

## ESP Output Frames(整合輸出)

“Our program is designed for…”
“The main need is…”
“This activity helps to…”
“The expected outcome is…”

## TEEMI Speaking Practice(升級版)

Task Flow
- **Reflect**(反思)
“What did you learn from the workshop?”
- **Identify**(需求)
“What problem do you want to address?”
- **Design**(設計)
“What is your program?”
- **Explain**(說明)
“How does it work?”

- **Program Design Framework**(這段是核心)
(建議你直接放投影片)
Green Healing Program Structure
- **Target Group**(誰)
- **Needs**(問題)
- **Intervention**(做什麼)
- **Outcome**(改變什麼)

加一層升級
- **Why this works**(為什麼有效)
(這是你原本沒有,但非常關鍵)

- **Critical Thinking Focus**(這週核心)
Students learn:
activity ≠ program
doing ≠ helping
design = purpose + structure + outcome

- **Teacher Language Upgrade**(你原本很好,我幫你升級)
你原句:
“Green healing is not just doing activities…”
升級👇
“Green healing is not just doing activities.
It is designing meaningful and responsible care.”

- **Assignment Upgrade**(幫你變更專業)
你原本很好,我幫你加一條👇
➕ 建議加:
● Why this program works(為什麼有效)
這會讓學生從:
✔ 描述 →
進入「解釋」
