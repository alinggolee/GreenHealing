# Week 14 ｜Green Healing Workshop II: Reflection and Program Design


## Learning Goals 學習目標

- By the end of this session, students will be able to:
- 在本節課結束時,學生將能夠:
- reflect critically on their green healing workshop experience.
- 對綠色療癒工作坊經驗進行批判性反思。
- summarize key knowledge and techniques learned from practice.
- 總結實作中學到的關鍵知識與技術。
- understand the concept of green healing program design.
- 理解綠色療癒方案設計的概念。
- develop a simple green healing program based on user needs.
- 根據使用者需求發展簡單的綠色療癒方案。

O — 導入(回顧與啟動)

快速回顧活動:

學生列出第13週的所有活動:

我們學到了什麼?

教師引入今日重點:
“From experience → to reflection → to program design”
「從經驗 → 到反思 → 到方案設計」

## D — Delivery

D — 講授(概念深化與影片學習)
Introduction to Green Healing Program Design:
介紹綠色療癒方案設計:
Identify target group (Who?)
確認目標對象(誰?)
Identify needs (What problem?)
確認需求(什麼問題?)

選擇方法(什麼活動?)

學生觀看參考影片:

荖葉及其文化與療癒意義

教師說明:

「當一個綠色療癒活動以目的與對象為核心設計時,它才會有意義。」

學生分組回答反思問題:

## R — Recap

R — 統整(綜合與作業說明)
Teacher summarizes key idea:
教師總結核心概念:
“Green healing is not just doing activities, but designing meaningful care.”
「綠色療癒不只是做活動,而是設計有意義的照護。」

Final Assignment Introduction 作業說明
Students will create a Green Healing Program Design
學生將設計一個綠色療癒方案
Assignment Requirements:
作業要求:
Target group (e.g., adolescent girls, university students)
目標對象(如青少女、大學生)
Health need (e.g., stress, menstrual discomfort)
健康需求(如壓力、經期不適)
Activity design (e.g., herbal drink workshop)
活動設計(如草本飲品工作坊)
Expected outcomes (e.g., relaxation, awareness, self-care ability)
預期成果(如放鬆、覺察、自我照護能力)

## ESP Vocabulary & Professional Usage 專業字彙與用法

Target Group: the people you design for
目標對象:你設計的服務對象
Needs Assessment: identifying problems or needs
需求評估:辨識問題或需求
Intervention: an action to improve a condition
介入:改善狀況的行動
Outcome: the result of an activity
成果:活動的結果
Reflection: thinking deeply about experience
反思:對經驗進行深入思考
Example 例句:
“Our program focuses on stress relief for university students.”
「我們的方案聚焦於大學生的壓力舒緩。」

我們如何將一個簡單活動轉化為有意義的方案?
What makes a green healing program effective?
什麼讓綠色療癒方案具有成效?

## TEEMI Speaking Practice 口說練習

Proposing a plan 提出方案:
“Our program is designed for students who experience stress.”
「我們的方案是為經歷壓力的學生設計的。」
Explaining purpose 說明目的:
“The goal is to improve emotional well-being through herbal drinks.”
「目標是透過草本飲品提升情緒健康。」
Reflecting 反思:
“From this workshop, I learned that healing can be simple and natural.”
「從這個工作坊中,我學到療癒可以是簡單且自然的。」
